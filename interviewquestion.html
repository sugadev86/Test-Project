JS Question

What is Javascript?

javascript is a client side scripting language.it can change both html and css.Also it can calculate,manipulate and validate data.

What is ES6?

ES6 is the standardized javascript.it is also called as ECMA script.ECMA script 2015 is the ES6 script.It has some new features like

  • Arrow function
  • rest and spread operator
  • destructuring
  • promises
  • etc ...
What is Js Engine?

Js Engine is a computer program for executes the js.js engines are developed by the webbrowser vendors.Each browser have one js engines they are

  • V8 - Google Chrome
  • SpiderMonkey - Mozilla
  • Javascriptcore - Apple's Safari
  • Chakra - Internet Explorer
  • Hermes - Android Apps

Global Scope and Local Shope in Js ?

In Js variable accessablity is called as scope.According to the place of declaring place scope will be categorized like

  • Global Scope

    In global scope variables are declared in outside of the function.That variable can accessable anywhere in the script.

  • Loca Scope

    In local scope variables are declared in inside of the function.That variable can accessable inside the function only.
