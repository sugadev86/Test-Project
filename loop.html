Loop

Repeated actions are called as loops.

Types of loop ?
  • while loop
  • do-while loop
while loop

In this loop have one conditions.it will runs till the conditions true otherwise it will stoped.

do-while loop

In this loop have the some function called do and also have condition.it will first run the do function then it will goes to the while condition if condition is false means it will stoped.
