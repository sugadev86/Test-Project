String Method

length

It is a function to find the total count of the letter in the string .

indexof

It is a function to find the position of the given string from the first place.

lastindexof

It is a function to find the position of the given string from the last place.

indexof and lastindexof with parameter

In Indexof and lastIndexof we can give a parameter means it will find the position of given string after the parameter value.

Search

Search is find the string and returns the location of the string

Slice

Slice is get the part of the string and also make it as new string.

Negative Slice

It is work like slice.the deference is it will count from the last.it means last string last letter is taking as zero and counts the values from right to left

single parameter slice

It is slice out positive values what we are giving, it will count the values from left to right returns the balance string and omit the counted. if it is slice out negative values what we are giving, it will count the values from left to right returns the counted string and omit the balance.

substring

It is works like slice but the difference is it will not accept the negative values.

single parameter substring

It is work like the slice single parameter.
